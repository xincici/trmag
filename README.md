**trmag**
Just a torrent seach tool.
trmag = torrent + magnet
use api from [torrentproject.com](http://torrentproject.com)

**usage**
```shell
npm install trmag -g
trmag -h
```

**advantage**
```shell
trmag -s 5 -n avatar    //搜索 avatar，并列出前5个
trmag avatar            //搜索 avatar，并列出前10个，默认
```
